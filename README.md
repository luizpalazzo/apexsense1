### HOW TO SETUP & RUN

1. Installation:

```
sudo apt install libx11-dev
git clone https://github.com/luizpalazzo/apexsense1.git
```

2. Compile code

```
cd myapex
g++ Main.cpp -lX11
```

3. Run and have fun 

```
sudo su
./a.out
```

If needed, change configuration on myapex.ini and offsets in Offsets.cpp
